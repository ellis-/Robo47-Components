About

Robo47 Components is a small library I use across my projects which extends the Zend Framework with components I often use. There are resources, validators, filters, an errorhandler, autoloaders and more.

Docs

An API-Documentation generated with doxygen can be found here, a small manual with examples will probably be done in the future and tutorials and more informations on the components will be published on my blog under the tag Robo47 Components.

Downloads

Get the code @github Robo47-Components

Requirements

Robo47 Components is build and test with Zend Framework 1.10.1 (Some bugs in prior versions can lead to problems), Doctrine 1.2.0+, HTMLPurifier 4.0.0, ezComponents 2009.1.2
My current setup for usage and testing is php 5.2.12, zend framework 1.10.1, phpunit 3.4.11, doctrine 1.2.1 on debian lenny and squeeze (development/testing).
